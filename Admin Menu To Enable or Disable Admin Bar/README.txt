== Description ==

Adds a new section to the Admin Side Toolbar (labeled: "Admin Bar") in which contains buttons that can enable or disable the admin bar for your WordPress Website. 
Properties are created in the WordPress database which retains the data for the last setting the user left the admin bar on.  

== Installation ==

1. Go to your WordPress Admin -> Plugins -> Add New.
2. Search for Admin Menu To Enable/Disable Admin Bar.
3. Install and Activate.

Or manually:

1. Download this folder and add it into your plugins directory. 
2. Activate the plugin through the 'Plugins' menu in WordPress.
